# MyFlask
Estrutura base para projetos Flask


##Start MyFlask
> clone o repositorio
> abri o terminal
> crie um ambiente virtual
    >$ python3 -m virtualenv -p python3 .venv
> ative o ambiente virtual
    >$ source .venv/bin/activate
> instale as bibliotecas
    >$ make install
> inicie o Flask
    >$ make run

> acesse no navegador o localhost
    >http://127.0.0.1:5000/
